Welcome to News Conglomerate!
___________________________________
Accumulates news from a variety of different sources and displays them. Can be easily configured to work for your own server with a REST API which queries pre-defined newspaper sets to be added to the user's list from the start.
Features:-
* Users can read different newspapers from all around the world, all through one application
* Users can add newspapers they like reading to favorites
* New newspapers can be added directly by the user or from the server
* Easily configurable to work on your own personalised server.
